# Admin

***Signup***
----
  Returns new Admin.

* **URL**

  /signup

* **Method:**

  `POST`
  
*  **URL Params**

    None

* **Data Params**

  * **Required:**
  ````
    {
      name: req.body.name,
      email: req.body.email,
      password: req.body.password,
    }
  ````

  * **Code:** 201 <br />

# Product

***Create***
----

* **URL**

  /product

* **Method:**

  `POST`
  
*  **URL Params**

    None

* **Data Params**

   **Required:**
  ````
    {
      name: req.body.name,
      price: req.body.price,
      stock: req.body.stock,
      image_Url: req.body.image_Url,
      category: req.body.category

  ````
